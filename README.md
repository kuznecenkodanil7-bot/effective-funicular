# ПРОТОКОЛ: СОБЕСЕДНИК

Desktop-игра по твоему промту. Это **не браузерная игра**: она запускается как обычное окно на ПК.

## Что уже реализовано

- консольный хоррор-пролог с Оператором;
- выборы в диалоге через `1 / 2 / 3` и `Enter`;
- ложное ветвление: разные ответы ведут к ключевому сообщению `Я ГОТОВ`;
- переход из консоли в основную игру;
- основная игра в виде top-down хоррора;
- медицинский комплекс с комнатами, коридорами и стенами;
- таблетки, которые сбрасывают таймер;
- звонящие телефоны;
- вопросы от Оператора с таймером ответа;
- тёмный силуэт, который появляется после ошибок и пропущенных звонков;
- силуэт не двигается, если игрок смотрит на него фонариком/мышью;
- память Оператора через save-файл: смерть и заражение сохраняются между запусками;
- победа через терминал `RESET`, когда собрано достаточно таблеток и отвечено на звонки;
- GitHub Actions для сборки `.exe` и Linux-бинарника.

## Управление

### Пролог

- `1 / 2 / 3` — выбрать вариант ответа.
- `Enter` — подтвердить выбранный вариант.
- `Space` — быстро допечатать текст.

### Основная игра

- `WASD` — ходить.
- `Shift` — бежать.
- `Мышь` — направление взгляда / фонарика.
- `E` — взаимодействие: взять таблетку, ответить на телефон, включить терминал.
- `R` — рестарт после смерти или победы.
- `Esc` — выход.

## Как запустить локально

Установи Python 3.11 или новее.

```bash
pip install -r requirements.txt
python main.py
```

На Windows можно просто запустить:

```bash
run.bat
```

## Как собрать через GitHub

1. Создай новый репозиторий на GitHub.
2. Загрузи туда все файлы из этой папки.
3. Открой вкладку **Actions**.
4. Запусти workflow **Build desktop game** вручную через **Run workflow** или сделай push в `main`.
5. После сборки открой завершённый workflow.
6. Внизу будет раздел **Artifacts**.
7. Скачай:
   - `ProtocolSobesednik-windows-latest` — Windows `.exe`;
   - `ProtocolSobesednik-ubuntu-latest` — Linux-бинарник.

## Как сделать, чтобы игра реально закрывалась при смерти

В `main.py` найди строку:

```python
REAL_CLOSE_ON_DEATH = False
```

Поменяй на:

```python
REAL_CLOSE_ON_DEATH = True
```

По умолчанию стоит `False`, потому что так удобнее тестировать: вместо настоящего закрытия показывается экран `CONNECTION LOST`.
